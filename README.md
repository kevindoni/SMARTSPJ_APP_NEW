# SmartSPJ

Aplikasi Pendamping ARKAS untuk Pengelolaan SPJ (Surat Pertanggungjawaban).

## 🚀 Fitur

- Dashboard monitoring keuangan
- Manajemen transaksi BKU
- Rekonsiliasi data
- Export kertas kerja
- Scraping data otomatis

## 📦 Instalasi

### Development

```bash
# Install dependencies
npm install

# Run development mode
npm run dev
```

### Build Production

```bash
# Build for production
npm run build:electron
```

## 🛠️ Tech Stack

- **Frontend:** React + Vite
- **Desktop:** Electron
- **Database:** SQLite (better-sqlite3)
- **Styling:** TailwindCSS
