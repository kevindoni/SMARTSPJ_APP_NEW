## 📌 About

SmartSPJ adalah aplikasi desktop yang dirancang sebagai **pendamping resmi penggunaan ARKAS** untuk membantu bendahara sekolah dalam menyusun dokumen **SPJ BOS** secara lebih cepat, rapi, dan akurat.

Berbeda dengan ARKAS, SmartSPJ berfokus pada:

- 📊 Analisis data keuangan
- 📑 Dokumen pertanggungjawaban
- 📚 Buku kas otomatis
- 📄 Laporan siap cetak
- 📈 Monitoring realisasi anggaran

> SmartSPJ **tidak mengubah database ARKAS.**
> Seluruh proses dilakukan menggunakan **mode Read-Only**.
